Псевдоклассы, псевдоэлементы и анимации в CSS: Полное руководство за 1,5 часа

Добро пожаловать! Сегодня мы разберем, как использовать псевдоклассы и псевдоэлементы в CSS, чтобы сделать сайты интерактивными, а также добавим анимации для создания динамичных эффектов. Эти инструменты помогут вам создавать современные интерфейсы без сложного кода.

Псевдоклассы — это состояния элементов (например, наведение курсора). Примеры: :hover, :focus.

Псевдоэлементы — это части элементов или добавленный контент. Примеры: ::before, ::after.

Анимации — это эффекты движения, которые можно комбинировать с псевдоклассами и псевдоэлементами.

История и назначение

Псевдоклассы появились в CSS1 (1996) с :hover и :active. В CSS3 добавились сложные селекторы вроде :nth-child(). Псевдоэлементы тоже эволюционировали: теперь они пишутся с ::. Анимации стали частью CSS3 с появлением transition и @keyframes.

Эти инструменты упрощают создание интерактивных сайтов без JavaScript.

Псевдоклассы: Подробный разбор

Псевдоклассы стилизуют элементы в зависимости от их состояния или позиции. Синтаксис: селектор:псевдокласс { свойства }.

Популярные псевдоклассы

  • :hover — при наведении курсора.
  • :active — при нажатии.
  • :focus — при фокусе.
  • :nth-child(n) — по порядку.
  • :first-child, :last-child — первый и последний.
  • :not() — исключение.

Примеры

1. Наведение на кнопку:

button:hover { background-color: #ff7043; }

2. Чередование строк:

li:nth-child(odd) { background: #e0f2f1; }

3. Фокус на поле ввода:

input:focus { border: 2px solid #0277bd; }

Псевдоэлементы: Подробный разбор

Псевдоэлементы стилизуют части элементов или добавляют контент. Синтаксис: селектор::псевдоэлемент { свойства }.

Популярные псевдоэлементы

  • ::before — перед элементом.
  • ::after — после элемента.
  • ::first-line — первая строка.
  • ::first Equals-letter — первая буква.
  • ::selection — выделенный текст.

Примеры

1. Декоративная стрелка:

h2::after { content: " ➔"; color: #d81b60; }

2. Большая первая буква:

p::first-letter { font-size: 2em; float: left; }

3. Кастомное выделение:

::selection { background: #ffca28; color: #333; }

Анимации в CSS

CSS позволяет создавать анимации с помощью transition (плавные изменения) и @keyframes (сложные эффекты). Они отлично сочетаются с псевдоклассами и псевдоэлементами.

Основы

  • transition — плавный переход (например, смена цвета за 0.3 секунды).
  • animation и @keyframes — задают последовательность изменений.

Примеры с псевдоклассами

1. Пульсация кнопки при наведении:

button:hover {
    transform: scale(1.1);
    transition: transform 0.3s ease;
}

2. Плавное появление текста при фокусе:

input:focus {
    border-color: #0277bd;
    transition: border-color 0.5s ease;
}

Примеры с псевдоэлементами

1. Анимация появления стрелки:

h2::after {
    content: " ➔";
    color: #d81b60;
    opacity: 0;
    transition: opacity 0.3s ease;
}
h2:hover::after {
    opacity: 1;
}

2. Пульсирующий круг перед текстом:

p::before {
    content: "●";
    color: #0277bd;
    margin-right: 5px;
    animation: pulse 1s infinite;
}
@keyframes pulse {
    0% { transform: scale(1); }
    50% { transform: scale(1.2); }
    100% { transform: scale(1); }
}

Интерактивный пример

Попробуйте создать анимацию для кнопки с псевдоклассами и псевдоэлементами:

Практические задания

1. Сделайте кнопку красной при наведении с плавным переходом за 0.5 секунды.

button { transition: background-color 0.5s ease; }
button:hover { background-color: red; }

2. Добавьте зеленую звездочку перед абзацем с анимацией вращения.

p::before {
    content: "★";
    color: green;
    margin-right: 5px;
    animation: spin 2s infinite;
}
@keyframes spin {
    100% { transform: rotate(360deg); }
}

3. Сделайте нечетные элементы списка синими с плавным появлением цвета.

li:nth-child(odd) {
    background-color: #0277bd;
    transition: background-color 0.4s ease;
}

4. Добавьте пульсирующий эффект кнопке при нажатии.

button:active {
    animation: pulse 0.5s;
}
@keyframes pulse {
    0% { transform: scale(1); }
    50% { transform: scale(1.1); }
    100% { transform: scale(1); }
}

5. Сделайте первую букву абзаца красной с плавным увеличением при наведении.

p::first-letter {
    color: red;
    font-size: 2em;
    transition: font-size 0.3s ease;
}
p:hover::first-letter {
    font-size: 2.5em;
}

6. Исключите последний элемент списка из анимации изменения цвета.

li:not(:last-child) {
    background-color: #ffca28;
    transition: background-color 0.3s ease;
}

Типичные ошибки

Ошибка: Забыть указать content в ::before.

Решение: Всегда добавляйте content: "".

Ошибка: Неправильный синтаксис @keyframes.

Решение: Убедитесь, что @keyframes идет перед использованием.

Ошибка: Использование transition без указания свойства.

Решение: Укажите, что анимировать: transition: transform 0.3s.

Викторина

1. Какой псевдокласс для нажатия?

:active

2. Как добавить анимацию вращения?

@keyframes с transform: rotate()

3. Как сделать плавный переход цвета?

transition: background-color 0.3s ease
