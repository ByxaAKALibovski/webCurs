CSS Grid, Flexbox и адаптивная верстка: Создаем современные макеты

Привет, ребята! Сегодня мы углубимся в CSS Flexbox и CSS Grid, чтобы научиться создавать крутые макеты для сайтов. А еще разберем, как сделать сайты удобными на телефонах и планшетах с помощью адаптивной верстки. Эти навыки помогут вам верстать всё — от портфолио до интернет-магазинов!

Flexbox — для гибких одномерных макетов (например, меню или карточки в ряд).

CSS Grid — для сложных двумерных макетов (например, сетка галереи).

Адаптивность — чтобы ваш сайт выглядел круто на любом устройстве.

Зачем это нужно?

Flexbox и Grid — основа современной верстки. Они заменили старые способы с float и position, упрощая создание сложных дизайнов. Адаптивная верстка делает сайты доступными для всех пользователей, будь то на большом мониторе или маленьком телефоне.

Примеры: сетка фотографий в соцсетях (Grid), горизонтальное меню (Flexbox), мобильная версия сайта (медиа-запросы).

Flexbox: Подробный разбор

Flexbox управляет расположением элементов в одном направлении (строка или столбец). Используйте display: flex на родительском элементе.

Ключевые свойства

  • justify-content — выравнивание по главной оси (center, space-between).
  • align-items — выравнивание по поперечной оси (center, stretch).
  • flex-wrap — перенос элементов (wrap, nowrap).
  • flex-grow/shrink — рост или сжатие дочерних элементов.

Примеры

1. Горизонтальное меню:

.nav {
    display: flex;
    justify-content: space-around;
    align-items: center;
}
.nav a { padding: 10px; }

2. Карточки с переносом:

.container {
    display: flex;
    flex-wrap: wrap;
    gap: 10px;
}
.card { flex: 1 1 200px; }

CSS Grid: Подробный разбор

Grid создает двумерные макеты с рядами и колонками. Используйте display: grid на родителе.

Ключевые свойства

  • grid-template-columns/rows — задают размеры колонок и строк.
  • gap — расстояние между ячейками.
  • grid-area — размещение элементов по именованным областям.
  • auto-fit/fill — адаптивные колонки.

Примеры

1. Простая сетка 3x3:

.grid {
    display: grid;
    grid-template-columns: repeat(3, 1fr);
    gap: 10px;
}
.item { background: #e0f2f1; padding: 15px; }

2. Макет сайта:

.layout {
    display: grid;
    grid-template-areas:
        "header header"
        "sidebar main"
        "footer footer";
    gap: 10px;
}
.header { grid-area: header; }
.sidebar { grid-area: sidebar; }
.main { grid-area: main; }
.footer { grid-area: footer; }

Адаптивная и респонсивная верстка

Адаптивность — это когда сайт подстраивается под разные экраны. Респонсивность — это плавное изменение дизайна.

Инструменты

  • @media — медиа-запросы для разных устройств.
  • rem, em, vw, % — относительные единицы.
  • min-width/max-width — ограничение размеров.

Примеры

1. Мобильное меню:

.nav {
    display: flex;
}
@media (max-width: 600px) {
    .nav {
        flex-direction: column;
    }
}

2. Адаптивная сетка:

.grid {
    display: grid;
    grid-template-columns: repeat(auto-fit, minmax(200px, 1fr));
    gap: 10px;
}

Интерактивный пример

Попробуйте настроить Flexbox или Grid для карточек:

Практические задания

1. Создайте горизонтальное меню с помощью Flexbox, которое центрирует элементы.

.menu {
    display: flex;
    justify-content: center;
    gap: 15px;
}
.menu a { padding: 10px; background: #0277bd; color: white; }

2. Сделайте сетку из 4 карточек с помощью Grid, с равными колонками.

Подсказка: Используйте repeat() и 1fr.

.grid {
    display: grid;
    grid-template-columns: repeat(4, 1fr);
    gap: 10px;
}
.card { background: #e0f2f1; padding: 15px; }

3. Адаптируйте Flexbox-контейнер, чтобы на мобильных он стал столбцом.

.container {
    display: flex;
    gap: 10px;
}
@media (max-width: 600px) {
    .container {
        flex-direction: column;
    }
}

4. Создайте адаптивную сетку с минимум 200px на колонку.

.grid {
    display: grid;
    grid-template-columns: repeat(auto-fit, minmax(200px, 1fr));
    gap: 10px;
}
.item { background: #b2ebf2; padding: 10px; }

5. Сделайте макет сайта с хедером, сайдбаром и контентом с помощью Grid.

.layout {
    display: grid;
    grid-template-areas: "header header" "sidebar content";
    grid-template-columns: 200px 1fr;
    gap: 10px;
}
.header { grid-area: header; background: #0277bd; }
.sidebar { grid-area: sidebar; background: #e0f2f1; }
.content { grid-area: content; background: #fff; }

6. Сделайте Flexbox-контейнер, где элементы растягиваются равномерно.

.container {
    display: flex;
    justify-content: space-between;
}
.item { flex: 1; background: #ffca28; margin: 5px; padding: 10px; }

Типичные ошибки

Ошибка: Забыть указать gap в Grid или Flexbox.

Решение: Всегда добавляйте gap: 10px для отступов.

Ошибка: Неправильный медиа-запрос.

Решение: Проверяйте max-width или min-width.

Ошибка: Использование Grid вместо Flexbox для простых макетов.

Решение: Flexbox для одномерных, Grid для двумерных.

Заключение
