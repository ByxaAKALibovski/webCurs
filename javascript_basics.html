Основы JavaScript: Делаем сайты интерактивными

Привет, ребята! Сегодня мы начинаем изучать JavaScript — язык, который оживляет сайты. С ним вы сможете менять содержимое страниц, реагировать на клики и создавать крутые эффекты. Это ваш первый шаг к созданию настоящих веб-приложений!

За 2 часа мы разберем, как работает JavaScript, научимся менять стили и текст на странице и добавим интерактивность. Погнали!

Что такое JavaScript и зачем он нужен?

JavaScript — это язык программирования, который работает в браузере. Если HTML — это скелет сайта, а CSS — его одежда, то JavaScript — это мышцы, которые заставляют страницу двигаться.

С помощью JS можно:

  • Менять текст и стили на странице.
  • Обрабатывать клики, ввод текста и другие действия.
  • Загружать данные с сервера (например, картинки).

Примеры: смена темы сайта, калькулятор, слайдер с фото.

Как работает JavaScript?

JS-код пишется в файле .js или внутри тега <script>. Он выполняется браузером, когда страница загружается или пользователь что-то делает.

Простейший пример:

console.log("Привет, мир!");

Открой консоль браузера (F12 → Console) и попробуй! Это команда выводит текст в консоль.

Основы синтаксиса

Переменные

Переменные хранят данные: текст, числа, цвета. Используйте let, const или var.

let name = "Аня";
const age = 15;
console.log(name, age); // Аня 15

Типы данных

Основные типы: строки ("текст"), числа (42), булевы (true/false).

let text = "Привет";
let number = 10;
let isCool = true;

Функции

Функции — это блоки кода, которые выполняются по вызову.

function sayHello(name) {
    console.log("Привет, " + name + "!");
}
sayHello("Саша"); // Привет, Саша!

Работа с DOM

DOM (Document Object Model) — это модель страницы, через которую JS взаимодействует с HTML и CSS.

Выбор элементов

Используйте querySelector для поиска элементов:

let button = document.querySelector(".my-button");
console.log(button); // Найдет элемент с классом my-button

Изменение стилей и текста

Меняйте текст с textContent, стили с style:

let title = document.querySelector("h1");
title.textContent = "Новый заголовок!";
title.style.color = "blue";

Обработка событий

Реагируйте на действия пользователя с addEventListener:

let button = document.querySelector(".my-button");
button.addEventListener("click", function() {
    alert("Кнопка нажата!");
});

Практические задания

1. Создайте кнопку, которая меняет текст заголовка h1 на "Привет, JS!"

let title = document.querySelector("h1");
let button = document.querySelector(".my-button");
button.addEventListener("click", function() {
    title.textContent = "Привет, JS!";
});

2. Сделайте так, чтобы при клике на карточку она становилась зеленой.

let card = document.querySelector(".card");
card.addEventListener("click", function() {
    card.style.backgroundColor = "green";
});

3. Напишите функцию, которая выводит в консоль "Клик!" при клике на кнопку.

let button = document.querySelector(".my-button");
button.addEventListener("click", function() {
    console.log("Клик!");
});

4. Создайте кнопку, которая добавляет новую карточку в контейнер.

let button = document.querySelector(".my-button");
button.addEventListener("click", function() {
    let newCard = document.createElement("div");
    newCard.className = "card";
    newCard.textContent = "Новая карточка";
    newCard.style.width = "100px";
    newCard.style.height = "100px";
    newCard.style.backgroundColor = "#ccc";
    document.querySelector(".container").appendChild(newCard);
});

5. Сделайте так, чтобы при вводе текста в input он появлялся в заголовке.

let input = document.querySelector(".my-input");
let title = document.querySelector("h1");
input.addEventListener("input", function() {
    title.textContent = input.value || "Введите текст";
});

Типичные ошибки

Ошибка: Код выполняется до загрузки страницы.

Решение: Используйте DOMContentLoaded или помещайте script в конец body.

Ошибка: Неправильный селектор в querySelector.

Решение: Проверяйте классы и теги, используйте консоль для отладки.

Ошибка: Забыть вызвать функцию.

Решение: Убедитесь, что функция вызывается, например, через событие.
